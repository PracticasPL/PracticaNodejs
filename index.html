**LINUX**

> sudo apt_get install python-software-properties
> sudo add-apt-repository ppa:chris-lea/node.js
> sudo apt-get update

**MAC**

> En caso de Mac deberemos utilizar el gestor de paquetes *homebrew* usando brew install nodejs.
